# Curated Challenges

This is a port of what I believe to be [my best mod for Slay the Spire 1](https://steamcommunity.com/sharedfiles/filedetails/?id=3337015720) with new challenges and features.

You can access the challenge runs from the main menu via a new button within the "Singleplayer" submenu.

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/1d27804d-e10e-4f53-afc6-104124ef4240" />

At the moment, there are **15** challenges with unique starting decks, relics, rules, and win conditions.

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/33e2d7b2-64d1-4f1d-a0de-a0a5593217ea" />

## NEW FEATURE: Shared Challenges

One of the most exciting new features of this mod compared to the Slay the Spire 1 version is **shared challenges**. These challenges appear in EVERY character's list and can be completed by anyone, with separate tracking for each character.

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/500a09f5-723d-4830-8115-661bc51cb835" />

## NEW FEATURE: Hidden Challenges

Some challenges may not show up on the challenges list until a prerequisite challenge is completed.

## Installation instructions
1. If there isn't already a `mods` folder located at `C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2` (or wherever your file path equivalent is), make one.
2. Put the latest release of this mod's .pck and .dll file in that folder.
3. Launch the game. Note that launching the game for the first time created new "modded" save folders, this is intentional and your unmodded saves have not been wiped.
